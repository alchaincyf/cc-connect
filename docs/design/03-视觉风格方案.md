# CC Connect 视觉风格方案

> 版本：v1.0 | 更新日期：2026-01-21

---

## 设计方向

CC Connect 是一个**开发者工具**，目标用户是 Claude Code 重度使用者。视觉风格应该：
- 专业、克制，不花哨
- 信息密度高，一目了然
- 终端风格元素，呼应 CLI 体验
- 深色模式优先（开发者偏好）

---

## 方案对比

| 方案 | 风格关键词 | 主色调 | 适合场景 | 参考 |
|------|-----------|--------|---------|------|
| A | 终端风格 | 绿色 + 深色 | 极客、硬核开发者 | iTerm2, Warp |
| B | 现代简约 | 蓝紫 + 深色 | 通用开发者 | Linear, Raycast |
| C | 苹果原生 | 系统色 + 浅/深 | 追求原生体验 | Apple Notes |

**推荐方案 B**：现代简约，平衡专业感和易用性。

## 核心组件设计

### 1. 状态指示器

```
运行中：  ●  绿色圆点 + 脉冲动画
等待输入：◉  橙色圆点 + 呼吸动画
空闲：    ○  灰色空心圆
错误：    ✕  红色叉号
断开：    ◌  灰色虚线圆
```

### 2. 会话卡片

```
┌─────────────────────────────────┐
│                                 │  背景：Surface
│  ● 会话名称                      │  圆角：12pt
│    状态 · 时间                   │  内间距：16pt
│    > 输出预览...                 │  阴影：cardShadow
│                                 │
└─────────────────────────────────┘

悬停/按下：背景加深 5%
```

### 3. 快捷回复按钮

```
┌────────┐     普通状态：
│   是   │     背景：Primary
└────────┘     文字：白色
               圆角：8pt
               高度：44pt

┌────────┐     次要按钮：
│   否   │     背景：Surface Secondary
└────────┘     文字：Text Primary
               边框：1pt Border
```

### 4. 输入框

```
┌─────────────────────────────────┐
│ 输入消息...                      │  背景：Surface Secondary
│                                 │  圆角：8pt
│                                 │  内间距：12pt
│                         [发送]  │  占位符：Text Tertiary
└─────────────────────────────────┘

发送按钮：
- 无内容：灰色，禁用
- 有内容：Primary，可点击
```

### 5. 输出区域

```
┌─────────────────────────────────┐
│ > Analyzing codebase...         │  背景：Terminal BG
│ > Found 15 files               │  文字：SF Mono 14pt
│ > Processing...                │  颜色：Terminal Text
│                                │  圆角：12pt
└─────────────────────────────────┘

特殊标记：
- 成功信息：Terminal Green
- 警告信息：Terminal Yellow
- 错误信息：Terminal Red
- 用户输入：Primary（紫色）
```

## App 图标设计方向

### 概念

结合"连接"和"代码"两个核心意象：
- 主体：两个相连的元素（代表手机与电脑的连接）
- 风格：简洁几何，渐变紫色
- 背景：深色（#0F0F0F）

### 草图

```
┌─────────────────┐
│                 │
│    ┌───┐ ⟷ ┌───┐│
│    │ > │   │ < ││
│    └───┘   └───┘│
│                 │
└─────────────────┘

方案：两个终端符号相连
配色：紫色渐变 (#6366F1 → #8B5CF6)
```

## 动效设计

### 1. 状态切换

```
运行中 → 等待输入：
- 圆点颜色渐变（绿 → 橙）0.3s ease
- 脉冲动画切换为呼吸动画

等待输入 → 运行中：
- 圆点颜色渐变（橙 → 绿）0.3s ease
- 添加"继续中"提示，0.5s 后消失
```

### 2. 消息到达

```
新消息：
- 从底部滑入，0.2s ease-out
- 自动滚动到底部

快捷按钮出现：
- 淡入 + 轻微上移，0.25s ease
```

### 3. 发送消息

```
发送按钮：
- 点击时轻微缩放 0.95
- 发送成功：输入框清空 + 轻微震动反馈
- 发送失败：输入框抖动 + 错误提示
```
