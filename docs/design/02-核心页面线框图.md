## 7. 组件规范

### 7.1 会话卡片

```
┌─────────────────────────────────┐
│ [状态指示] 会话名称              │
│   状态文字 · 时间               │
│   最后一条输出预览...           │
└─────────────────────────────────┘

状态指示颜色：
- ● 绿色 #34C759 = 运行中
- ◉ 橙色 #FF9500 = 等待输入
- ○ 灰色 #8E8E93 = 空闲
- ✕ 红色 #FF3B30 = 错误
```

### 7.2 快捷回复按钮

```
┌────────┐  ┌────────┐  ┌────────┐
│   是   │  │   否   │  │  继续  │
└────────┘  └────────┘  └────────┘

规格：
- 高度：44pt
- 圆角：8pt
- 间距：12pt
- 字体：17pt Medium
```

### 7.3 输入框

```
┌─────────────────────────────┬────────┐
│ 输入消息...                 │  发送  │
└─────────────────────────────┴────────┘

规格：
- 高度：44pt（单行），最高120pt（多行）
- 圆角：8pt
- 发送按钮：当有内容时高亮
```
